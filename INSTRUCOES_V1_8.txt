O que entrou:

2. Sidebar desktop mantida.
3. Botão Sair do LMS continua ativo.
4. Tela /admin/lojas para cadastrar, editar e inativar lojas.
5. Tela /admin/cargos para cadastrar, editar e inativar cargos/funções.

7. Admin principal com atalhos para lojas, cargos e grupos.
8. Ajustes globais para reduzir estouros horizontais em celular.

Como aplicar:
1. Substitua a pasta next-app usando o ZIP completo.
2. Rode o SQL: next-app/supabase/v1.8_organization_mobile.sql no Supabase.
3. Faça git push para a Vercel.
4. Teste no celular: /, /cursos, /admin, /admin/lojas, /admin/cargos, /admin/grupos, /admin/relatorios.
